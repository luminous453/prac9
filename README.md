# Практическая работа №9

### Инструкция
1. Клонируем проект с файлами 
    ```shell
    git clone https://github.com/luminous453/prac9
   ```
2. Запускаем Докер
    ```shell
    docker-compose up --build 
   ```
3. Сайт админа
   ```shell
   http://localhost:3000
   ``` 
4. Пользовательская страничка
    ```shell
   http://localhost:8080
   ```
